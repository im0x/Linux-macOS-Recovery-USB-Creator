# Linux-macOS-Recovery-USB-Creator
Create a bootable macOS Recovery USB from Linux (Debian/Ubuntu, Fedora, Arch). Pulls Apple's BaseSystem.dmg for Monterey (12.6)+ via fetch-macOS.py. Direct dd; Bypasses flaky Internet Recovery (e.g., Sierra stalls). Tested Nov 2025 on Ubuntu 25.10/sudo-rs, Fedora 42, Arch. ~3GB output; legal/public mirrors. Scripts + troubleshooting in README.
